# ⚡ Consultor de Precios del Mercado Eléctrico - CENACE

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://tu-usuario.streamlit.app)

Aplicación interactiva para consultar y visualizar los precios zonales del Sistema Interconectado Nacional (SIN) de México a través de los webservices del CENACE.

## 🚀 Características

- 📊 Consulta de precios MDA (Día Adelantado) y MTR (Tiempo Real)
- 🗺️ 101 zonas disponibles del SIN
- 📅 Selección de rango de fechas
- 📈 Gráficos interactivos (líneas y barras)
- 📋 Tabla detallada con todos los datos
- 📊 Estadísticas y análisis comparativo
- 💾 Descarga de datos en CSV

## 🔧 Requisitos

- Python 3.8+
- Streamlit
- Pandas
- Requests
- Plotly

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/tu-repositorio.git
cd tu-repositorio

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar la aplicación
streamlit run app.py
